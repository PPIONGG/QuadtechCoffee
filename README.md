# ระบบเว็บไซต์ ทำงานร่วมกับ json-server

## ติดตั้ง json server

npm i json-server

## ต้องรันการทำงานพร้อมกัน ทั้ง node และ json server

- npm start

- json-server --watch db.json --port 4000
